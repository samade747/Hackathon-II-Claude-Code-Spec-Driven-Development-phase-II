## Core Components

- `Navbar`
  - Shows app title.
  - Shows login/logout or user avatar (from Better Auth).
- `TodoList`
  - Renders a list of tasks with title, status, priority, tags, due date.
- `TodoItem`
  - Single task row.
  - Buttons: Complete/Toggle, Edit, Delete.
- `TodoForm`
  - Inputs: title, description, priority, tags, due date, recurrence fields.
  - Used for both create and edit modes.
- `TodoFilters`
  - Controls for search, status, priority, tag, sort.

## Style Constraints

- Use Tailwind CSS utilities.
- Dark theme by default (slate/sky colors).
- Responsive: works well on mobile and desktop.
